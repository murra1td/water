# Arizona Water Resources

An investigative research project tracing Arizona's water supply — from source to tap — and building tools to make that journey visible.

## Goals

- Build a deep understanding of Arizona's water resources
- Trace the path water takes from its source to a residential address
- Produce a public explainer (blog, slides, or video)
- Build an address-lookup tool that maps the water path for any Arizona address
